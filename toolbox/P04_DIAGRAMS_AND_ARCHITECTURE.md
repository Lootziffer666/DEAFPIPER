# Diagram Prompt Cheatsheet

# Diagram Prompt Cheatsheet -- Flowcharts, Mindmaps & Architecture

This guide contains ready-to-use prompts to generate diagrams such as
flowcharts, mindmaps, system architecture diagrams, ER diagrams,
timelines, and more.

------------------------------------------------------------------------

# 1. Flowchart Prompts (Processes)

1. Create a flowchart for an online order process from browsing to
 delivery.
2. Visualize the employee onboarding process in a company.
3. Build a flowchart showing a customer support ticket workflow.
4. Create a decision-based flowchart for approving or rejecting loan
 applications.
5. Show the bug tracking lifecycle from report to resolution.
6. Create a flowchart for a CI/CD pipeline.
7. Visualize the process of publishing a blog post.

------------------------------------------------------------------------

# 2. Mindmap Prompts (Ideas & Brainstorming)

8. Create a mindmap for digital marketing strategy.
9. Build a mindmap for launching a startup.
10. Visualize all components of a SaaS product ecosystem.
11. Create a mindmap for productivity techniques.
12. Generate a mindmap for learning data science.
13. Build a mindmap for personal goal planning.

------------------------------------------------------------------------

# 3. Software Architecture Prompts

14. Visualize a microservices architecture with API gateway, auth
 service, and databases.
15. Create a cloud architecture diagram for a scalable web app.
16. Show a serverless architecture using API gateway, lambda functions,
 and storage.
17. Visualize a Kubernetes-based microservices system.
18. Create an architecture diagram for a real-time chat application.
19. Design a high-level architecture for an AI recommendation system.

------------------------------------------------------------------------

# 4. Sequence Diagram Prompts (Interactions)

20. Show the login process between user, frontend, API, and database.
21. Visualize a payment processing interaction with third-party gateway.
22. Create a sequence diagram for password reset flow.
23. Show the interaction between microservices in an order processing
 system.
24. Visualize the API authentication flow using OAuth.

------------------------------------------------------------------------

# 5. Database / ER Diagram Prompts

25. Create an ER diagram for an e-commerce platform.
26. Model a database for a university system (students, courses,
 professors).

28. Create a database schema for a social media platform.

------------------------------------------------------------------------

# 6. Timeline Prompts

30. Create a timeline for launching a new product.
31. Visualize the history of artificial intelligence development.
32. Show the phases of a software development project.

------------------------------------------------------------------------

# 7. Pie Chart Prompts

34. Show the distribution of marketing budget across channels.
35. Visualize company revenue by product category.
36. Show time allocation across daily work tasks.
37. Display survey results by response percentage.

------------------------------------------------------------------------

# Pro Tip

Good prompts include: - The diagram type - Key entities or components -
Relationships or interactions - Optional decisions or conditions

Example:

"Create a flowchart for a user registration process including email
verification, validation errors, and successful account creation."
